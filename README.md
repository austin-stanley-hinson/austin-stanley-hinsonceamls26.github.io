This repository contains my personal CEAMLS SAIRI Summer Research portfolio site, built with Jekyll and deployed on GitHub Pages.

## About Me

I am Austin Hinson, a Computer Science and Applied Mathematics student at Colby College. My interests include software engineering, machine learning, artificial intelligence, and building systems that can create meaningful real-world impact.

This portfolio documents my work during the CEAMLS SAIRI Summer Research Program, including project progress, reflections, and key deliverables.

## Research Focus

My project is titled:

**AI-Driven Discovery of Drug Candidate Compounds to Slow Cognitive Decline in Alzheimer's Disease**

The work centers on using computational methods and QSAR-based machine learning workflows to support early-stage drug discovery. Current workflow areas include:

- molecular dataset curation and preprocessing
- descriptor extraction for QSAR modeling
- model training and comparison (Decision Trees, Random Forests, Logistic Regression, KNN)
- evaluation with accuracy, precision, recall, F1 score, and confusion matrices

## Local Development

```bash
bundle install
bundle exec jekyll serve --livereload
```

Open `http://localhost:4000` to preview the site locally.

To preview all themes:

```bash
bin/preview-all
```
